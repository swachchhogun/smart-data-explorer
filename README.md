# Smart Data Explorer

An interactive data analysis dashboard built with **Python, Streamlit, Pandas, and Plotly** that allows users to upload CSV datasets and explore insights through dynamic visualizations.

---

## Live Demo

🚀 https://smart-data-explorer.streamlit.app

---

## Features

- Upload and analyze CSV datasets
- Interactive dataset filtering
- Distribution analysis
- Category comparison charts
- Correlation heatmap
- Trend analysis over time
- Automatic dataset insights

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly

---

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

---

## Project Purpose

This project demonstrates interactive **Exploratory Data Analysis (EDA)** using Python.  
Users can upload datasets and quickly visualize patterns, trends, and correlations through an interactive dashboard.
